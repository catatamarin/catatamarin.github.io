Chen Karlsonn - pedant light, favourite things: Esta original lampara colgante tiene un difusor de luz transparente en el que puedes crear cualquier escena, con la ayuda de pequeños objetos, como juguetes, flores, imágenes, etc. Esta va a resaltar tus objetos bajo una fuente de luz.   1. Frank Gehry- Silla Wiggle:

Frank Gehry desarrollo una serie de muebles llamada Easy Edges, con la que buscaba darle una nueva dimensión estética a un material tan corrientes como el cartón, es confortable, solida y resistente, Tiene unas medidas de 61 x 35 x 87 y la fábrica Vitra. La silla fue diseñada en el año 1972 mas como una escultura, pero comenzó a comercializarla en el año 1997.

Propuestas

En esta parte desarrolle tres mobiliarios hibidros:

El primero es una mesa movil, pensada especificamente para lugares pequeños para aprovechar de mejor forma el especio y que no exista problema para movilizarla de manera sensilla.

El segundo es un sillon de terraza que en la parte de el respaldo se encuentran unos maceteros, tambien para ahorrar espacio.

La tercera opción es un escritorio el cual tiene una luz incluida y en vez de una mesa común de base tiene una pizarra para poder tomar apuntes de manera rapida.
